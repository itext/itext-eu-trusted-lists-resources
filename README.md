# itext-eu-trusted-lists-resources

This repository contains the resources required to use when validating eu lotl's.
This code is written in java in the `java` folder and then autoported using the sharpen maven plugin.

The C# code is generated by using the `mvn clean install -Pwith-sharpen` command.

### Current publication we support

* [C/2026/1944](https://eur-lex.europa.eu/eli/C/2026/1944/oj)

### Project structure

```
- java                  # Java project that bundles the correct certificates 
- sharp                 # C# project that bundles the correct certificates 
```
